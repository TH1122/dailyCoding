array.sort() 메서드를 사용할 때, 숫자를 오름차순이나 내림차순으로 정렬 하려면 아래와 같이 함수를 넣어줘야한다는것을 잊지 말고 기억하자. 

```jsx
[1,2,123,4,321,3,23].sort();
// -> [1, 123, 2, 23, 3, 321, 4]

[1,2,123,4,321,3,23].sort((a,b) => a-b);  // 오름차순
// -> [1, 2, 3, 4, 23, 123, 321]

[1,2,123,4,321,3,23].sort((a,b) => b-a);  // 내림차순
// -> [321, 123, 23, 4, 3, 2, 1]
```
